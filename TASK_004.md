# 任务：添加 Telegram 快捷按钮和菜单栏

## 需求

1. **Bot 命令菜单** - Telegram 内置命令列表
   - 用户输入 `/` 时显示可用命令
   - 每个命令有简短描述

2. **快捷按钮** - Inline Keyboard
   - 管理员操作后显示快捷按钮（查看/编辑/删除）
   - 客户查询后显示快捷按钮（续费/修改信息）

3. **主菜单** - Reply Keyboard（可选）
   - 常用功能快捷入口

## 需要修改的文件

- `app/bot/handlers.py` - 添加命令菜单配置
- `app/bot/keyboards.py` - 新建键盘布局
- `app/bot/menus.py` - 新建菜单配置

## 验收标准

1. ✅ 输入 `/` 显示命令列表
2. ✅ 每条命令有清晰描述
3. ✅ 关键操作有快捷按钮
4. ✅ 按钮布局合理，不拥挤
